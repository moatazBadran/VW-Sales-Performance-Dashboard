# VW-Sales-Performance-Dashboard Developed By Moataz Badran
Sales Performance Report is made to display Sales Insights across Regions and Models over Time

## Overview
The **Volx Sales Performance Dashboard** is a Power BI project designed to analyze sales performance across multiple dimensions, including Models, Fuel_Types, DealerShip, Customer_Segments, time, and Countries.

This dashboard provides actionable insights to support business decision-making, identify growth opportunities, and monitor key performance indicators (KPIs).

---

## 🎯 Objectives
- Track overall sales and profitability
- Analyze performance by Units, Models, and customer segments
- Monitor trends over time
- Identify top-performing and underperforming Regions
- Support data-driven decision-making

---

## 📂 Dataset Description
The dataset follows a **star schema model** and includes:

### 🔹 Fact Table
- **FactSales**
  - Units Sold
  - Unit Price
  - Unit Cost
  - Revenue
  - Profit

### 🔹 Dimension Tables
- **DimDate** → Date, Month, Year
- **Model** → Models details
- **CustomerSegment** → Customer Segments
- **FuelType** → Product categories
- **DealerShip** → Dealer ship
- **Country** → Counties and Regions

---

## 📊 Key KPIs
- 💰 Total Sales
- 📦 Total Units Sold
- 📈 Total Profit
- 📊 Profit Margin %
- 🧾 Average Unit Revenue
- 🥇 Top Performing Units
- 👥 Sales per Models
